la potenciacion es una operacion matematica usada para multiplicar un 
numero por si mismo varias veces, es una forma abreviada de hacer
multiplicasiones seguidas en la que intervienen 2 elementos que son 
La base que es el numero que se multiplicara
El exponente que indica cuantas veces se multiplicara

3^2 = 9

siendo el 3 la base y 2 el exponente 

PROPIEDADES DE LA POTENCIACION

Producto de potencia de igual base.
la multiplicasion de potencias de igual base es igual a la misma
potencia elevada la suma de sus exponentes 

3^6 x 3^2 = 3^6+2

Conciente de potencia de igual base.
cuando se dividen potencias de iguak base se deja la misma base y 
se restan los exponentes

10^4 รท 10^3 = 10^4-3

Potencia de una potencia.
cuando una potensia se eleva a otro exponente se mantiene la misma 
base y se multiplican los exponentes 

5^3^4 = 5^3x4

Potencia de un producto.
cuando un prosucto se eleva a un exponente se eleva cada 
factor del mismo

(3x2)^4 = 3^4 x 2^4
